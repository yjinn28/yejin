# HTML, CSS, JS 기초

## HTML

> HTML : Hyper Text Markup Language - 마크업(표시) 언어

> 웹페이지의 구조를 표시

> 웹페이지의 콘텐츠를 표시

## HTML Elements

> Element : 요소 - 의미적

> Tag : 태그 - 기술적

> 문법(Syntax)
> 
> <>로 감싸줌
> 
> 소문자로 사용
> 
> 시작태그와 종료태그 세트로 구성됨
```
<tagname>contents</tagname>
```
> 
> 예외) 시작태그만 존재하는 경우 : 빈 요소(Empty Element)
> 
> 포함관계(Nested)
```
<body>
  <div>
    text
  </div>
</body>
```

### HTML Attribute

> HTML 속성
> HTML 요소의 부가 정보
> 속성name = "속성value"

```
<a href="http://www.naver.com">naver</a>
```
